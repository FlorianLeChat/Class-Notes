Notes

HTML (Hyper Text Markup Language) est un langage de balisage hypertexte, qui est utilisé pour créer les pages web. Il est composé des éléments sémantiques et structurels, qui définissent comment des parties du document (les titres, les paragraphes, les listes, les images etc.) doivent être présentées dans le navigateur.

L'unité structurelle de base d'une page Web est un élément HTML, pour la définition de lequel on utilise les balises HTML. Les balises HTML sont utilisés pour définir les structures du contenu sur la page Web. À l'aide de balises, nous marquons le contenu et faisons comprendre au navigateur comment afficher quelque partie du contenu (par exemple, texte, image, lien hypertexte, etc.). Les balises ne sont cependant pas affichées dans le navigateur. Les balises HTML sont placées entre crochets angulaires <balise>, par exemple <html>. La plupart des balises HTML sont appariées (par exemple, <p> </p>), c’est-à-dire, il sont composés des deux parties : le tag d’ouverture (<p>) et le tag de fermeture (</p>). Les balises seules n’ont pas de tag de fermeture. Par exemple, le tag <img> est seul, qui est utilisé pour spécifier des images.

L'élément <!DOCTYPE> indique que le document actuel appartient à un type spécifique de HTML. Il existe plusieurs versions de HTML, et pour que le navigateur ne soit pas confus et affiche le bon standard, il doit le spécifier. Il existe plusieurs types de « DOCTYPE », pour chaque version du langage HTML existe le sien. Voici un tableau récapitulant les versions HTML utilisées par le développeur ou non :
