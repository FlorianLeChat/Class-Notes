Notes

Les ordinateurs qui se connectent au Web sont appelés des clients et des serveurs. Les clients correspondent aux appareils des utilisateurs connectés sur Internet (par exemple, votre ordinateur connecté par Wi-Fi ou votre téléphone connecté sur le réseau mobile) et aux navigateurs web (comme par exemple, Firefox ou Chrome). Les serveurs sont des ordinateurs (le plus souvent des serveurs dans des centre de données) qui stockent des pages web, des sites ou des applications diverses et variées. Lorsqu'un appareil client souhaite accéder à une page web, une copie de la page est téléchargée depuis le serveur vers le client (sous la forme d'une chaîne de caractères), la machine utilisée affiche alors le contenu dans le navigateur de l'utilisateur.

Lorsque vous saisissez une adresse web dans votre navigateur :
  • le navigateur demande au DNS l'adresse réelle du serveur contenant le site.
  • le navigateur envoie une requête HTTP au serveur pour lui demander d'envoyer une copie du site web au client. Ce message, et les autres données envoyées entre le client et le serveur, sont échangés par l'intermédiaire de la connexion internet.
  • si le serveur accepte la requête émise par le client, le serveur envoie un message « 200 OK » au client qui signifie : « Pas de problème, tu peux consulter ce site web, le voici ». Ensuite le serveur commence à envoyer les fichiers du site web au navigateur sous forme d'une série de petits morceaux nommés « paquet ».
  • le navigateur assemble les différents morceaux pour recomposer le site web en entier puis l'affiche sur votre écran.
